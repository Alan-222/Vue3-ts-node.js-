# Vue3-ts-node.js-
一个前端vue3+ts，后端node.js的后台管理系统项目
### 技术栈

#### 前端

- 语言：`typeScript`

- 前端框架：`vue 3`
- 脚手架：`vite`
- 路由：`vue-router 4`
- 状态管理：`vuex 4`
- CSS 预编译处理：`sass`
- 网络请求工具：`axios`
- 前端 UI 框架：`element-plus 2`

#### 后端

- 语言：`javaScript`
- 运行环境：`node.js 14  `
- 后端开发框架：`express 4  ` 
- 数据库：`mysql`

### 运行项目

#### 前端

文件夹：vue3+ts demo
命令：1. npm install
      2. npm run dev
#### 后端
文件夹：node.js_apiServer
命令 1.npm install
     2.npm run start
